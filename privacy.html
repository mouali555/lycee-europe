Confidentialité

Cette page décrit le fonctionnement général des données. Pour une conformité complète (RGPD), il faut adapter selon votre organisation.

Données collectées

  • Authentification : compte Google (UID, nom, photo).
  • Messages : texte, images envoyées (si tu utilises l’upload).
  • Préférences locales : thème/sons (stockées dans ton navigateur).

Utilisation de l’IA

  • @ia est déclenché uniquement lorsque tu l’appelles.
  • Le prompt est transmis à un service IA pour générer une réponse.
  • La réponse est publiée dans la room comme un message “IA”.

Suppression

  • Tu peux supprimer tes messages (icône poubelle sur tes bulles).
  • Les modérateurs peuvent supprimer en cas d’abus.

Sécurité

  • Accès filtré par invite et vérifié côté serveur.
  • Les endpoints sensibles utilisent un token Firebase (Bearer).
